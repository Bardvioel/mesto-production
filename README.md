# Mesto Russia

## 📖 О проекте

**Mesto** — интерактивная веб-приложение, где пользователи могут делиться фотографиями мест, которые они посетили. Это итоговая работа от Яндекс Практикума, демонстрирующая навыки работы с современным JavaScript, асинхронными запросами, валидацией форм и взаимодействием с API.

### ✨ Основные возможности

- 📝 **Редактирование профиля** — изменение имени, описания и аватара
- 📸 **Добавление карточек** — публикация фотографий с названием мест
- ❤️ **Взаимодействие с контентом** — лайки карточек
- 🗑️ **Управление контентом** — удаление собственных карточек
- 🔍 **Просмотр изображений** — увеличение фотографий в полноэкранном режиме
- 📊 **Статистика** — просмотр аналитики по пользователям и карточкам
- 📱 **Адаптивный дизайн** — корректное отображение на всех устройствах

Frontend

## 🎯 Функциональные особенности

### 🔐 Валидация форм
- Проверка ввода в реальном времени
- Кастомные сообщения об ошибках
- Блокировка кнопки отправки при невалидных данных

### 🖼️ Модальные окна
1. **Редактирование профиля** - изменение имени и описания
2. **Добавление новой карточки** - публикация фотографий
3. **Обновление аватара** - замена изображения профиля
4. **Подтверждение удаления** - подтверждение перед удалением карточки
5. **Просмотр изображения** - полноэкранный просмотр фотографий
6. **Статистика пользователей** - аналитика по активности

### 📊 Система статистики
При клике на логотип открывается панель статистики с:
- Общим количеством карточек
- Датами создания первой и последней карточек
- Количеством уникальных пользователей
- Максимальным количеством карточек от одного пользователя
- Списком всех пользователей
